El proyecto es un MVC que toma los datos de la BD de mongodb atlas,
obtiene la informacion y la muestra en react. 
se corre con npm start primero se tiene que correr en el back 
porque inicia en el puerto 3000 y despues en el front, solo muestra 
los datos de la grafica.
El back se corre con: 
npm run-script start-dev
El front con: 
npm start

Debes ir a : 
http://localhost:3001/donut
Donde se mostrara todas las graficas.
